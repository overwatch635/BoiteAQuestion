Modification Directe dans le Code

Pour adapter cette application à votre filière ou classe, modifiez le fichier script.js et cherchez la section "DONNÉES DE BASE". Vous pouvez y ajouter, modifier ou supprimer :

  • Les thèmes de questions
  • Les questions et réponses
  • Les cours et définitions

Toutes les modifications seront automatiquement reflétées dans l'application après rechargement de la page.
